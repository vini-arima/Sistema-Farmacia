# Sistema-Farmacia

Projeto feito linguagem JAVA, Nesse projeto utilizo os conceitos de arquitetura de software 
e programação orientada a objetos. 
\
Escolhi a arquitetura MVC ( Model , View , Controller ) para estrutura de pastas e para persistência de dados utilizarei XML. 

Além disso utilizarei Java Swing para criar Interface Gráfica (GUI).
